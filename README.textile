h1. About

'Minify' is a plugin to minify JavaScript and/or CSS code and combine it to one file. This will increase performance of website.

So you don't need to do HTTP request for dozens files, all can be combined in one file on the fly.

h1. Usage

1. Install the minify plugin to your Frog CMS plugins dir:

@cd /path/to/frog/plugins/@
@git clone git://github.com/dknight/frog_minify.git@

2. Activate minify plugin
!http://www.dmitri.me/misc/minify.png!

3. Set the settings in the 
!http://www.dmitri.me/misc/minify2.png!

3. Usage in PHP
@<?php
$files = array(
    $_SERVER['DOCUMENT_ROOT'] . '/public/javascripts/jquery-1.3.2.min.js',
    $_SERVER['DOCUMENT_ROOT'] . '/public/javascripts/jquery.validate.min.js',
    $_SERVER['DOCUMENT_ROOT'] . '/public/javascripts/jquery.form.js',
    $_SERVER['DOCUMENT_ROOT'] . '/public/javascripts/jquery.scrollTo-min.js',
    $_SERVER['DOCUMENT_ROOT'] . '/public/javascripts/frog-' . $_SESSION['language'] . '.js',
    $_SERVER['DOCUMENT_ROOT'] . '/public/javascripts/frog.js'
);

$js_minify = Minify::factory('js');
?>
<script type="text/javascript" src="<?php echo $js_minify->minify($files, true);?>"></script>
@
